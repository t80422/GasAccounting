# 如何在主選單加入銀行月結餘額重整工具

## 方法一：在現有選單中加入（建議）

如果 `frmMain` 有 MenuStrip 控制項，可以在「工具」或「維護」選單中加入：

### 1. 在 Designer 中加入選單項目

```vb
' 在 frmMain.Designer.vb 中
' 假設已有 MenuStrip 名為 MenuStrip1

' 加入「工具」選單（如果沒有的話）
Me.ToolStripMenuItem工具 = New System.Windows.Forms.ToolStripMenuItem()
Me.ToolStripMenuItem工具.Text = "工具(&T)"

' 加入「重整銀行月結餘額」選單項目
Me.ToolStripMenuItem重整銀行餘額 = New System.Windows.Forms.ToolStripMenuItem()
Me.ToolStripMenuItem重整銀行餘額.Text = "重整銀行月結餘額(&R)"
Me.ToolStripMenuItem重整銀行餘額.Name = "ToolStripMenuItem重整銀行餘額"

' 加入到選單
Me.ToolStripMenuItem工具.DropDownItems.Add(Me.ToolStripMenuItem重整銀行餘額)
Me.MenuStrip1.Items.Add(Me.ToolStripMenuItem工具)
```

### 2. 在 frmMain.vb 中加入事件處理

```vb
Private Sub ToolStripMenuItem重整銀行餘額_Click(sender As Object, e As EventArgs) Handles ToolStripMenuItem重整銀行餘額.Click
    OpenBankBalanceRecalculator()
End Sub
```

## 方法二：在 TabPage 中加入按鈕

如果想在某個 TabPage（例如「基本設定」或「系統維護」）中加入按鈕：

### 1. 在 Designer 中加入按鈕

```vb
' 在適當的 TabPage 中加入按鈕
Me.btnRecalculateBankBalance = New System.Windows.Forms.Button()
Me.btnRecalculateBankBalance.Location = New System.Drawing.Point(20, 20)
Me.btnRecalculateBankBalance.Size = New System.Drawing.Size(180, 40)
Me.btnRecalculateBankBalance.Text = "重整銀行月結餘額"
Me.btnRecalculateBankBalance.Font = New Font("微軟正黑體", 10, FontStyle.Bold)
Me.btnRecalculateBankBalance.BackColor = Color.LightCoral

' 加入到 TabPage（例如 tpBasicSet）
Me.tpBasicSet.Controls.Add(Me.btnRecalculateBankBalance)
```

### 2. 在 frmMain.vb 中加入事件處理

```vb
Private Sub btnRecalculateBankBalance_Click(sender As Object, e As EventArgs) Handles btnRecalculateBankBalance.Click
    OpenBankBalanceRecalculator()
End Sub
```

---

## 方法三：使用快捷鍵（進階）

可以設定快捷鍵來快速開啟工具：

### 在 frmMain.vb 中加入鍵盤事件

```vb
Private Sub frmMain_KeyDown(sender As Object, e As KeyEventArgs) Handles MyBase.KeyDown
    ' Ctrl + Shift + R 開啟重整工具
    If e.Control AndAlso e.Shift AndAlso e.KeyCode = Keys.R Then
        OpenBankBalanceRecalculator()
        e.Handled = True
    End If
End Sub
```

## 方法四：在「登出」TabPage 中加入（最簡單）

因為您的 frmMain 有 `tpLogOut` TabPage，可以在這裡加入維護工具：

### 1. 在 tpLogOut 中加入按鈕

```vb
' 在 frmMain.Designer.vb 的 InitializeComponent 中
Me.btnRecalculateBankBalance = New System.Windows.Forms.Button()
Me.btnRecalculateBankBalance.Location = New System.Drawing.Point(20, 100)
Me.btnRecalculateBankBalance.Size = New System.Drawing.Size(200, 50)
Me.btnRecalculateBankBalance.Text = "🔧 重整銀行月結餘額"

Me.btnRecalculateBankBalance.BackColor = Color.FromArgb(255, 224, 192)
Me.btnRecalculateBankBalance.Cursor = Cursors.Hand

Me.tpLogOut.Controls.Add(Me.btnRecalculateBankBalance)
```

### 2. 加入事件處理

```vb
' 在 frmMain.vb 中
Private Sub btnRecalculateBankBalance_Click(sender As Object, e As EventArgs) Handles btnRecalculateBankBalance.Click
    OpenBankBalanceRecalculator()
End Sub
```

## 完整範例：在現有程式碼中加入

### 步驟 1：找到適當的位置

在 `frmMain.Designer.vb` 中找到適當的 TabPage 或區域

### 步驟 2：宣告控制項

```vb
' 在 frmMain.Designer.vb 的類別宣告區域加入
Friend WithEvents btnRecalculateBankBalance As Button
```

### 步驟 3：初始化控制項

```vb
' 在 InitializeComponent() 方法中加入
Me.btnRecalculateBankBalance = New System.Windows.Forms.Button()
Me.btnRecalculateBankBalance.Location = New System.Drawing.Point(20, 20)
Me.btnRecalculateBankBalance.Size = New System.Drawing.Size(200, 50)
Me.btnRecalculateBankBalance.Text = "重整銀行月結餘額"
Me.btnRecalculateBankBalance.Font = New Font("微軟正黑體", 11, FontStyle.Bold)
Me.btnRecalculateBankBalance.BackColor = Color.LightSalmon
Me.btnRecalculateBankBalance.ForeColor = Color.DarkRed
Me.btnRecalculateBankBalance.Cursor = Cursors.Hand
Me.btnRecalculateBankBalance.FlatStyle = FlatStyle.Flat

' 加入到適當的容器（例如 tpBasicSet 或 tpLogOut）
Me.tpLogOut.Controls.Add(Me.btnRecalculateBankBalance)
```

### 步驟 4：加入事件處理（已完成）

```vb
' 在 frmMain.vb 中（已經有 OpenBankBalanceRecalculator 方法）
Private Sub btnRecalculateBankBalance_Click(sender As Object, e As EventArgs) Handles btnRecalculateBankBalance.Click
    OpenBankBalanceRecalculator()
```

## 權限控制（選擇性）

如果需要限制只有特定角色才能使用此功能：

```vb
Private Sub frmMain_Load(sender As Object, e As EventArgs) Handles MyBase.Load
    ' ... 其他初始化程式碼 ...
    
    ' 檢查權限
    If Not HasPermission("系統維護") Then
        btnRecalculateBankBalance.Visible = False
        ' 或
        btnRecalculateBankBalance.Enabled = False
    End If
End Sub

Private Function HasPermission(permissionName As String) As Boolean
    ' 根據您的權限系統實作
    ' 例如：檢查當前使用者的角色
    Return True ' 暫時返回 True
End Function
```

## 注意事項

1. **UI 設計**
   - 建議使用警告色系（橘色、紅色）提醒使用者此為維護工具
   - 加入適當的圖示增加辨識度

2. **位置選擇**
   - 建議放在不常用的位置（如系統維護 TabPage）
   - 避免誤觸

3. **權限控制**
   - 建議只開放給管理員或特定角色使用

4. **使用說明**
   - 可以在按鈕旁加入說明文字
   - 或使用 ToolTip 提供簡短說明
